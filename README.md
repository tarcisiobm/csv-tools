# csv-tools

Client-side web application for CSV file manipulation. No backend required — runs entirely in the browser.

[access here](https://tarcisiobm.github.io/csv-tools/)

## Tools

- **Compare** — Compare headers between a base (reference) CSV and a target CSV. Fix the target to match the base structure.
- **Filter** — Filter rows by column value with equal/not-equal conditions and case-sensitivity toggle.
- **Replace** — Bulk-replace values across selected columns.

## Features

- Auto-detects delimiter (`;`, `,`, `\t`, `|`)
- All processing happens locally in the browser
- No dependencies or build step
